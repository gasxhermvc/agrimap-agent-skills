## Boundary ที่สำคัญ

### Memory ไม่ควรเก็บ

- Transcript การคุยทั้งหมด
- Tool output ดิบ
- Prompt ทุก invocation
- รายละเอียด step-by-step ทุกอย่างที่ Agent ทำ
- Hidden chain-of-thought
- ข้อมูลที่ไม่มี source หรือไม่รู้ว่ายัง valid อยู่หรือไม่

### Tasks ไม่ควรเก็บ

- Permanent knowledge ที่ถูกใช้ข้ามหลายงานโดยไม่มีการ promote
- Raw telemetry
- Prompt template ที่ copy มาจากที่อื่น
- Changelog ของระบบทั้งโครงการ

### Logs ไม่ควรเป็น

- Project documentation
- Memory ที่ Agent โหลดทุกครั้ง
- พื้นที่เก็บ prompt template
- พื้นที่เก็บผลลัพธ์แบบที่มนุษย์ต้องไล่อ่านเอง
