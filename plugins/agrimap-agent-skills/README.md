# AgriMap Agent Skills plugin package

This directory is the self-contained Codex and Claude plugin package for the official `agrimap-agent-skills` umbrella.

Author workflow rules, references, assets, and scripts only in:

```text
skills/agrimap-agent-skills/
```

The following plugin content is generated by `npm run sync`:

- `skills/agrimap-agent-skills/`: byte-for-byte mirror of the authored umbrella skill;
- `skills/agm-*/`: thin operation aliases generated from `config/operations.json`;
- `docs/` and `examples/`: published copies of the repository sources;
- provider hooks, commands, and synchronized manifest versions.

Do not edit generated copies directly. Fix the canonical source, run `npm run sync`, then run `npm test` and `npm run validate`.
