// upload step
step1: client > upload shapefile to agmws-file-management-netcore with url https://domain.com/agmws-gateway/agmws-data-management/content/upload
step2: agmws-data-management-netcore > call http://agmws-file-management:5000/api/agmws-file-management/file/upload

// publish step
step1: client > publisher mapservice to https://domain.com/agmws-gateway/agmws-layer-management/publish/mapservice/{content_id}
step2: agmws-layer-management-netcore call http://agmws-fil-management:5000/api/file-management/file/{file_id} to download file
step3: insert record to database and call GeoProcessing to publish mapservice
